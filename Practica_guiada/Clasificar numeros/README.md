Tendremos un sistema que debe clasificar el resultado de una evaluación. Como dato de entrada tendremos un entero y como resultado un String de la clasificación de la siguiente forma:

93 - 100 --> Excelente
85 - 92 --> Sobresaliente
75 - 84 --> Distinguido
60 - 74 --> Bueno
00 - 59 --> Desaprobado
